title: "Kestrel 2.0 Hybrid: Heated door mirror inoperative, replace mirror assembly"

---

# Condition

Some Kestrel 2.0 Hybrid vehicles may exhibit an inoperative heated exterior door mirror, with no mirror glass defrost function on cold-weather start. On diagnostic scan the Body Control Module stores DTC B2477, indicating an open or high-resistance condition in the heated mirror element circuit of door mirror assembly DRMR-7040. Customers typically report the concern intermittently before the element fails completely. See [Kestrel Body Service Manual :: Body Diagnostic Trouble Code Reference](doc://man-kestrel-body#diagnosis/body-diagnostic-trouble-code-reference) for the full definition of B2477.

# Cause

The root cause is a manufacturing defect in early-production DRMR-7040 mirror assemblies, where the heating element solder joint fractures under repeated thermal cycling and opens the circuit. Once the element opens, the BCM detects loss of load and sets DTC B2477. The fault is internal to the mirror assembly and cannot be repaired in the field. Refer to [Kestrel Body Service Manual :: Exterior Mirror Sub-System](doc://man-kestrel-body#exterior-mirror-sub-system) and [Kestrel Body Service Manual :: Body Electrical System Overview](doc://man-kestrel-body#body-electrical-system-overview) for circuit details.

# Repair Procedure

Replace the affected door mirror assembly DRMR-7040 with a revised-production unit, following [Kestrel Body Service Manual :: Door Mirror Assembly Replacement](doc://man-kestrel-body#exterior-mirror-sub-system/door-mirror-assembly-replacement). After installation, perform the BCM configuration procedure PROC-BCM-CONFIG so the module relearns the heated mirror load and clears DTC B2477; see [Kestrel Electrical Service Manual :: BCM Configuration Procedure](doc://man-kestrel-electrical#body-control-module-sub-system/bcm-configuration-procedure). If configuration does not complete, diagnose per [Kestrel Body Service Manual :: Module Configuration Failure Diagnosis](doc://man-kestrel-body#diagnosis/module-configuration-failure-diagnosis) before returning the vehicle. Verify the repair by confirming the mirror element warms and no codes return on a cold-soak retest.
