title: Vega lost communication with body control module U0140

# Condition

Affected Vega vehicles may exhibit inoperative interior lighting, intermittent dash warning lamps, and a no-crank condition with diagnostic trouble code U0140 stored. U0140 indicates lost communication with the body control module, and other modules on the network may set companion codes when the BCM drops off the bus. The instrument cluster may flicker or go dark while the fault is active. See the [Vega Electrical Service Manual :: Lost Communication with BCM and Cluster](doc://man-vega-electrical#network-communication-diagnostic-codes/lost-communication-with-bcm-and-cluster) for the full symptom list.

# Cause

The U0140 condition is most often caused by a corrupted configuration in the body control module, part number BCM-9920, following a low-voltage event during cranking. When the module fails to complete its wake-up handshake, neighboring modules report it as missing from the CAN bus. The role of this module is described in the [Vega Electrical Service Manual :: Body Control Module](doc://man-vega-electrical#electrical-system-overview/body-control-module). Network-level code definitions are listed in the [Vega Electrical Service Manual :: Network Communication Diagnostic Codes](doc://man-vega-electrical#network-communication-diagnostic-codes).

# Repair Procedure

Before replacing hardware, reconfigure the body control module BCM-9920 by following PROC-BCM-CONFIG as detailed in the [Vega Electrical Service Manual :: Body Control Module Configuration Procedure](doc://man-vega-electrical#bcm-sub-system-service/body-control-module-configuration-procedure). If U0140 persists after configuration, inspect the bus wiring per the [Vega Electrical Service Manual :: CAN Bus Communication Diagnosis Procedure](doc://man-vega-electrical#network-and-harness-sub-system-service/can-bus-communication-diagnosis-procedure). Clear all stored codes and confirm the BCM responds to a network scan before returning the vehicle. General module locations are documented in the [Vega Electrical Service Manual](doc://man-vega-electrical).
