id: tsb-26-104
title: Kestrel downstream oxygen sensor aging triggering catalyst efficiency code
area: bulletins
---

# Condition

Affected Kestrel vehicles may set DTC P0420 (Catalyst System Efficiency Below Threshold, Bank 1) with the MIL illuminated and no driveability complaint. The code typically appears after extended mileage as the post-catalyst signal begins to track the pre-catalyst signal too closely. Freeze-frame data usually shows steady-state cruise conditions at operating temperature when the fault matures. Background on this code is provided in [Kestrel Emissions Service Manual :: Catalyst Efficiency Faults](doc://man-kestrel-emissions#emissions-system-diagnosis/catalyst-efficiency-faults).

# Cause

The most common cause is aging of the downstream oxygen sensor, part number O2D-5120, whose slowed switching response is misinterpreted by the monitor as reduced catalyst storage. In many cases the catalytic converter CAT-5500 is still functional, and a lazy O2D-5120 alone trips P0420. Sensor function and placement are described in [Kestrel Emissions Service Manual :: Upstream and Downstream Oxygen Sensors](doc://man-kestrel-emissions#oxygen-sensing/upstream-and-downstream-oxygen-sensors), and converter theory in [Kestrel Emissions Service Manual :: Catalytic Converter (Bank 1 and Bank 2)](doc://man-kestrel-emissions#catalytic-conversion/catalytic-converter-bank-1-and-bank-2).

# Repair Procedure

Replace the downstream oxygen sensor O2D-5120 following PROC-O2-SENSOR, applying anti-seize sparingly to the threads and torquing to specification. After replacement, clear DTC P0420 and run the catalyst monitor to completion to confirm the code does not return before condemning the converter. The detailed steps are in [Kestrel Emissions Service Manual :: Oxygen Sensor Replacement Procedure](doc://man-kestrel-emissions#oxygen-sensing/oxygen-sensor-replacement-procedure), within the broader [Kestrel Emissions Service Manual :: Oxygen Sensing](doc://man-kestrel-emissions#oxygen-sensing) section.

# Catalyst Replacement (If Required)

If P0420 returns after the O2D-5120 sensor is replaced and verified, the catalytic converter CAT-5500 has degraded and must be replaced following PROC-CAT. Inspect for upstream causes such as misfire or oil consumption before fitting a new CAT-5500 to avoid repeat failure. The replacement procedure is documented in [Kestrel Emissions Service Manual :: Catalytic Converter Replacement Procedure](doc://man-kestrel-emissions#catalytic-conversion/catalytic-converter-replacement-procedure).
