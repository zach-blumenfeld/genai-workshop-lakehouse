id: tsb-22-322
title: "Kestrel 1.5T: P0420 catalyst efficiency below threshold from degraded upstream O2 sensor"
area: bulletins
---

# Condition

Kestrel 1.5T vehicles may present with an illuminated MIL and stored DTC P0420 (Catalyst System Efficiency Below Threshold, Bank 1) without any accompanying driveability complaint. The code typically sets after extended highway operation once the monitor completes. Before condemning the catalytic converter, technicians must verify the upstream and downstream oxygen sensor switching behavior, as a lazy upstream sensor frequently triggers this code. See [Kestrel Emissions Service Manual :: Catalyst Efficiency Faults](doc://man-kestrel-emissions#emissions-system-diagnosis/catalyst-efficiency-faults) for the full P0420 diagnostic tree.

# Cause

In affected vehicles the root cause is a degraded upstream oxygen sensor, part number O2U-5100, whose slowed switching rate causes the PCM to misinterpret converter oxygen storage and falsely set P0420. The catalytic converter CAT-5500 is rarely the true cause and should only be replaced after the upstream sensor has been ruled out. Review oxygen sensor operation in [Kestrel Emissions Service Manual :: Upstream and Downstream Oxygen Sensors](doc://man-kestrel-emissions#oxygen-sensing/upstream-and-downstream-oxygen-sensors) and converter function in [Kestrel Emissions Service Manual :: Catalytic Converter (Bank 1 and Bank 2)](doc://man-kestrel-emissions#catalytic-conversion/catalytic-converter-bank-1-and-bank-2).

# Repair Procedure

Replace the upstream oxygen sensor O2U-5100 following PROC-O2-SENSOR, taking care to apply anti-seize only to the threads and to route the harness clear of exhaust heat. Step-by-step removal and torque values are given in [Kestrel Emissions Service Manual :: Oxygen Sensor Replacement Procedure](doc://man-kestrel-emissions#oxygen-sensing/oxygen-sensor-replacement-procedure). Background on the oxygen sensing subsystem is available in [Kestrel Emissions Service Manual :: Oxygen Sensing](doc://man-kestrel-emissions#oxygen-sensing). Do not replace CAT-5500 under this bulletin unless post-repair verification still indicates low catalyst efficiency.

# Verification

After replacing the upstream sensor, clear P0420 and complete a drive cycle to allow the catalyst monitor to run to completion. Confirm the upstream sensor now switches rapidly and the downstream sensor remains relatively steady, indicating adequate oxygen storage. If P0420 returns after the monitor completes, proceed with full converter evaluation per [Kestrel Emissions Service Manual :: Emissions System Diagnosis](doc://man-kestrel-emissions#emissions-system-diagnosis). Document the switching data captured during verification in the repair order.
