# Condition

Owners of the Marlin EV may report a complete loss of cabin airflow with the HVAC fan failing to respond at any speed setting. When this condition is present, the climate control module sets DTC B1352 (blower motor circuit malfunction) and stores it as a current fault. Begin diagnosis by confirming the active code and reviewing freeze-frame data per the [Marlin EV Hvac Service Manual :: Blower Motor Circuit Fault Diagnosis](doc://man-marlin-ev-hvac#hvac-system-diagnosis/blower-motor-circuit-fault-diagnosis). The cross-platform [Heron Hvac Service Manual :: Blower Motor Circuit Code](doc://man-heron-hvac#diagnostic-trouble-codes/blower-motor-circuit-code) describes an analogous code definition and may aid technicians familiar with that platform.

# Cause

The root cause of DTC B1352 has been traced to intermittent open or high-resistance conditions in the blower motor BLW-7200 power feed circuit, frequently at the connector interface. Heat cycling degrades the terminal crimp, raising circuit resistance until the module can no longer command the blower and logs B1352. Refer to the [Marlin EV Hvac Service Manual :: HVAC System Diagnosis](doc://man-marlin-ev-hvac#hvac-system-diagnosis) for the full circuit description, and consult the [Heron Hvac Service Manual :: Blower Motor and Cabin Air Filter](doc://man-heron-hvac#hvac-system-overview/blower-motor-and-cabin-air-filter) for component context shared across the HVAC family.

# Repair Procedure

Replace the affected blower motor BLW-7200 following PROC-BLOWER, taking care to inspect and reseat the connector before final assembly. The complete removal and installation steps are documented in the [Heron Hvac Service Manual :: HVAC Blower Motor Replacement Procedure](doc://man-heron-hvac#air-distribution-sub-system-service/hvac-blower-motor-replacement-procedure), which applies to the air distribution sub-system covered in [Heron Hvac Service Manual :: Air Distribution Sub-System Service](doc://man-heron-hvac#air-distribution-sub-system-service). After installing the new BLW-7200, clear DTC B1352, command the blower through all speeds, and verify cabin airflow is restored. For broader system reference see the [Heron Hvac Service Manual](doc://man-heron-hvac).
