# Condition

On the Kestrel, technicians may encounter a module configuration failure following replacement of a headlamp assembly, with the body control module setting DTC B2477. The vehicle may exhibit warning lamps, inoperative exterior lighting, or a failed end-of-line configuration. Confirm the stored code against the [Kestrel Body Service Manual :: Body Diagnostic Trouble Code Reference](doc://man-kestrel-body#diagnosis/body-diagnostic-trouble-code-reference) and review the [Kestrel Body Service Manual :: Module Configuration Failure Diagnosis](doc://man-kestrel-body#diagnosis/module-configuration-failure-diagnosis) before proceeding.

# Cause

DTC B2477 is set when the BCM cannot validate the configuration data for the newly installed headlamp assembly HLMP-7020, typically because the part was fitted without the required module write. The mismatch between the stored configuration and the installed HLMP-7020 triggers the fault and disables affected lighting outputs. The [Kestrel Body Service Manual :: Body Electrical System Overview](doc://man-kestrel-body#body-electrical-system-overview) describes how the BCM authenticates lighting components, and the [Kestrel Body Service Manual :: Diagnosis](doc://man-kestrel-body#diagnosis) outlines the supporting checks.

# Repair Procedure

Install headlamp assembly HLMP-7020 per PROC-HEADLAMP as detailed in the [Kestrel Body Service Manual :: Headlamp Assembly Replacement Procedure](doc://man-kestrel-body#lighting-sub-system/headlamp-assembly-replacement-procedure), then perform the module configuration per PROC-BCM-CONFIG. Run the configuration using the [Kestrel Electrical Service Manual :: BCM Configuration Procedure](doc://man-kestrel-electrical#body-control-module-sub-system/bcm-configuration-procedure); the equivalent cross-platform routine is documented in the [Vega Electrical Service Manual :: Body Control Module Configuration Procedure](doc://man-vega-electrical#bcm-sub-system-service/body-control-module-configuration-procedure). After configuring HLMP-7020, clear DTC B2477, cycle the lighting, and verify the code does not return. For lighting sub-system context see the [Kestrel Body Service Manual :: Lighting Sub-System](doc://man-kestrel-body#lighting-sub-system).
