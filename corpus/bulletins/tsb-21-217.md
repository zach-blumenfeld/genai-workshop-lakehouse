title: Falcon system voltage low P0562 from failing alternator

# Condition

Falcon owners may report a battery warning lamp, dimming headlights, and hard starts, with diagnostic trouble code P0562 stored in the engine control module. P0562 indicates system voltage low, meaning measured charging voltage has fallen below the expected threshold during engine operation. In severe cases the vehicle may stall once battery reserve is depleted. The code is defined in the [Falcon Charging Service Manual :: System Voltage Codes](doc://man-falcon-charging#diagnostic-trouble-codes/system-voltage-codes).

# Cause

The P0562 condition is caused by a failing alternator, part number ALT-8810, whose internal voltage regulator no longer maintains rated output under load. Once charging output drops, the battery BAT-1200 discharges and cannot support electrical demand, reinforcing the low-voltage fault. The alternator is described in the [Falcon Charging Service Manual :: Alternator](doc://man-falcon-charging#charging-system-overview/alternator), and the battery in the [Falcon Charging Service Manual :: Battery](doc://man-falcon-charging#charging-system-overview/battery). Related control-circuit faults are listed in the [Falcon Charging Service Manual :: Generator Control Circuit Code](doc://man-falcon-charging#diagnostic-trouble-codes/generator-control-circuit-code).

# Repair Procedure

Confirm the fault by following PROC-ALT as described in the [Falcon Charging Service Manual :: Charging System Diagnosis Procedure](doc://man-falcon-charging#charging-system-diagnosis-sub-system/charging-system-diagnosis-procedure), then replace alternator ALT-8810. Load-test battery BAT-1200 and replace it per the [Falcon Charging Service Manual :: Battery Replacement Procedure](doc://man-falcon-charging#battery-sub-system-service/battery-replacement-procedure) if it fails to hold charge. After repair, verify charging voltage is within specification and clear P0562. General component locations appear in the [Falcon Charging Service Manual](doc://man-falcon-charging).
