title: "Osprey 3.5 V6: Torque converter clutch stuck off sets P0741 with shudder"
area: bulletins
---

# Condition

Some Osprey 3.5 V6 vehicles may exhibit a noticeable shudder during light-throttle acceleration between 35 and 50 mph, often accompanied by an illuminated MIL. Scan of the transmission control module reveals diagnostic trouble code P0741, indicating the torque converter clutch circuit is stuck in the off position. Customers typically describe the sensation as a brief vibration or "rumble strip" feel as the converter clutch attempts to engage. The condition is intermittent at first and tends to worsen as fluid degrades.

# Cause

The root cause is sticking of the torque converter clutch (TCC) apply solenoid, part number TSOL-9520, which fails to fully command converter lockup and triggers P0741. Contaminated or oxidized transmission fluid passing through filter TFLT-9500 accelerates solenoid valve deposit buildup and restricts hydraulic apply pressure. In affected units the solenoid bore varnishing prevents the valve from holding commanded position under load. See the [Osprey Transmission Service Manual :: Transmission System Overview](doc://man-osprey-transmission#transmission-system-overview) for a description of the hydraulic apply circuit and the [Osprey Transmission Service Manual :: Fluid and Filter Sub-System](doc://man-osprey-transmission#fluid-and-filter-sub-system) for filter specifications.

# Diagnostic Procedure

Begin by following the full [Osprey Transmission Service Manual :: Transmission Fault Diagnosis Procedure](doc://man-osprey-transmission#diagnosis/transmission-fault-diagnosis-procedure) (PROC-TRANS-DIAG) to confirm P0741 as a current code rather than a stored history fault. Inspect fluid level and condition per the [Osprey Transmission Service Manual :: Fluid Level and Condition Inspection](doc://man-osprey-transmission#fluid-and-filter-sub-system/fluid-level-and-condition-inspection), noting any burnt odor or dark coloration of filter TFLT-9500. Cross-reference the symptom against the [Osprey Transmission Service Manual :: Diagnostic Trouble Code Reference](doc://man-osprey-transmission#diagnosis/diagnostic-trouble-code-reference) and verify apply pressure using the [Osprey Transmission Service Manual :: Fluid Pressure Diagnosis](doc://man-osprey-transmission#diagnosis/fluid-pressure-diagnosis). If commanded TCC apply pressure does not hold within spec, proceed to the repair procedure.

# Repair Procedure

Replace the TCC apply solenoid, part number TSOL-9520, following the [Osprey Transmission Service Manual :: Shift Solenoid Pack Replacement Procedure](doc://man-osprey-transmission#shift-control-sub-system/shift-solenoid-pack-replacement-procedure) (PROC-SHIFT-SOL). Drain and refill the transmission and renew the filter as part of the same service to remove varnish contamination. After installation, command TCC apply with a scan tool and road test to confirm P0741 does not reset and the shudder is eliminated. Refer to the [Osprey Transmission Service Manual :: Shift Control Sub-System](doc://man-osprey-transmission#shift-control-sub-system) for solenoid pack torque values and connector routing.
