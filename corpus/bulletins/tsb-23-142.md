# Condition

Summit owners may report the ABS warning lamp illuminating intermittently or remaining on, often with the traction control light, after vehicle start or during low-speed maneuvers. A scan of the ABS module returns DTC C0035 (Left Front Wheel Speed Sensor Circuit). The condition is frequently aggravated by road salt, water intrusion, or harness chafing at the front wheel well. For an orientation on how the sensors integrate with the module, see the [Summit Abs Service Manual :: Wheel Speed Sensors](doc://man-summit-abs#abs-system-overview/wheel-speed-sensors).

# Cause

The fault is traced to the left front wheel speed sensor WSS-3300, where corrosion or an open in the sensor element or its pigtail produces an erratic or absent signal and sets C0035. In some vehicles the connector is intact but the sensor internal coil has failed, requiring sensor replacement rather than harness repair. Use the diagnostic flow in [Summit Abs Service Manual :: Wheel Speed Sensor Circuit Codes](doc://man-summit-abs#diagnostic-trouble-codes/wheel-speed-sensor-circuit-codes) to isolate the sensor from the circuit before ordering parts. General system context is found in the [Summit Abs Service Manual :: ABS System Overview](doc://man-summit-abs#abs-system-overview).

# Repair Procedure

Replace sensor WSS-3300 following procedure PROC-WSS, ensuring the mounting bore is clean and the air gap is within specification. The complete torque values and routing are given in the [Summit Abs Service Manual :: Wheel Speed Sensor Replacement Procedure](doc://man-summit-abs#wheel-speed-sensor-sub-system-service/wheel-speed-sensor-replacement-procedure); clear C0035 and verify with a road test above 25 km/h. If the harness is suspect rather than the sensor, return to the diagnostic tree in [Summit Abs Service Manual :: Diagnostic Trouble Codes](doc://man-summit-abs#diagnostic-trouble-codes) before condemning WSS-3300.
