# Condition

Affected Ridgeline HD 6.7 TD vehicles may set DTC P2002 (Diesel Particulate Filter Efficiency Below Threshold). Drivers may report illuminated check engine and DPF warning lamps, reduced power from a forced derate, and frequent regeneration attempts. Confirm the code against the [Emissions Diagnostic Trouble Code Reference](doc://man-ridgeline-hd-emissions#diagnosis/emissions-diagnostic-trouble-code-reference) and review the [DPF Efficiency Faults](doc://man-kestrel-emissions#emissions-system-diagnosis/dpf-efficiency-faults) topic for shared diagnostic logic before proceeding.

# Cause

The condition is caused by accumulated ash loading and partial substrate degradation in diesel particulate filter DPF-5180, which lowers measured filtration efficiency below the calibrated threshold and sets P2002. Excessive short-trip operation or repeatedly interrupted regenerations accelerate this loading. See the [Diesel Particulate Filter](doc://man-kestrel-emissions#diesel-particulate-filter) topic and the [Emissions System Overview](doc://man-ridgeline-hd-emissions#emissions-system-overview) for filter construction and efficiency monitoring details.

# Repair Procedure

Begin with the emissions system checks in [Diagnosis](doc://man-ridgeline-hd-emissions#diagnosis) and attempt a service regeneration per PROC-DPF-REGEN using the [Diesel Particulate Filter Service and Forced Regeneration](doc://man-ridgeline-hd-emissions#diesel-particulate-filter-sub-system/diesel-particulate-filter-service-and-forced-regeneration) procedure. If efficiency does not recover after forced regeneration, replace diesel particulate filter DPF-5180 following the [DPF Service and Regeneration](doc://man-kestrel-emissions#diesel-particulate-filter/dpf-service-and-regeneration) guidance. After repair, reset the ash counter, clear P2002, and verify a completed regeneration cycle on a road test.
