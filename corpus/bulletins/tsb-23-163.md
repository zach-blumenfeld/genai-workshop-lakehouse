# Condition

Vega owners may report multiple simultaneous symptoms such as inoperative interior lighting, non-functioning power locks, and an unresponsive instrument cluster. A network scan returns DTC U0140 (Lost Communication With Body Control Module) stored in one or more modules sharing the CAN bus. The condition may be intermittent and is often triggered by moisture intrusion or a marginal connector at the body control module. For module context, see the [Vega Electrical Service Manual :: Body Control Module](doc://man-vega-electrical#electrical-system-overview/body-control-module).

# Cause

The cause is loss of communication to body control module BCM-9920, resulting from an internal fault, corroded connector pins, or an open in the CAN bus serving the module, which sets U0140 across the network. Before replacing BCM-9920, confirm bus integrity and power and ground at the module, because a wiring or termination fault will reproduce the same code. The symptom pattern and module isolation steps are detailed in [Vega Electrical Service Manual :: Lost Communication with BCM and Cluster](doc://man-vega-electrical#network-communication-diagnostic-codes/lost-communication-with-bcm-and-cluster). General architecture is in the [Vega Electrical Service Manual :: Electrical System Overview](doc://man-vega-electrical#electrical-system-overview).

# Repair Procedure

Begin with procedure PROC-COMMS-DIAG to isolate the U0140 condition between the module and the harness, following [Vega Electrical Service Manual :: CAN Bus Communication Diagnosis Procedure](doc://man-vega-electrical#network-and-harness-sub-system-service/can-bus-communication-diagnosis-procedure). If BCM-9920 is confirmed faulty, replace it and complete the [Vega Electrical Service Manual :: Body Control Module Configuration Procedure](doc://man-vega-electrical#bcm-sub-system-service/body-control-module-configuration-procedure) to program the new module, then clear U0140 and verify all affected functions. The complete code reference is in the [Vega Electrical Service Manual :: Network Communication Diagnostic Codes](doc://man-vega-electrical#network-communication-diagnostic-codes).
