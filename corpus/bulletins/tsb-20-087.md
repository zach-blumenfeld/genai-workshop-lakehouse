title: Updated Catalyst Monitor Calibration

# Condition

Falcon and Osprey vehicles may set catalyst code P0420 with no driveability
complaint and a catalyst that tests within spec. An over-sensitive monitor
calibration is the cause in these cases.

# Cause

The previous ECM-9901 calibration flags marginal catalyst readings too
aggressively, producing false P0420 reports.

# Repair Procedure

Apply the updated ECM-9901 calibration per
[ECM Software Updates](doc://man-fal-3#emissions/ecm-software-updates) before
replacing any CAT-5500 catalytic converter. Confirm there is no active misfire
first — see [Catalyst Efficiency](doc://man-fal-3#emissions/catalyst-efficiency).
