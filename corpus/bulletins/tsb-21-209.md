title: Summit left front wheel speed sensor fault C0035 disabling ABS
area: bulletins
---

# Condition

Affected Summit vehicles may store DTC C0035 (left front wheel speed sensor circuit) and disable the ABS and stability control systems, illuminating the associated warning lamps. The fault often appears intermittently at low speed before becoming permanent. Drivers may notice the warning lamps clear after a key cycle only to return during the next drive. Confirm the code and the affected circuit using [Summit Abs Service Manual :: Wheel Speed Sensor Circuit Codes](doc://man-summit-abs#diagnostic-trouble-codes/wheel-speed-sensor-circuit-codes).

# Cause

DTC C0035 is set when the left front wheel speed sensor WSS-3300 produces an intermittent or open signal, typically due to a degraded internal element or chafed harness near the sensor. Because the ABS controller cannot trust the affected wheel signal, it disables antilock function as a safety measure. See [Summit Abs Service Manual :: Wheel Speed Sensors](doc://man-summit-abs#abs-system-overview/wheel-speed-sensors) for sensor construction and [Summit Abs Service Manual :: ABS System Overview](doc://man-summit-abs#abs-system-overview) for how the controller responds to a lost signal.

# Repair Procedure

Replace the left front wheel speed sensor WSS-3300 following PROC-WSS, routing and securing the harness to prevent future chafing. Clear C0035 and verify ABS function with a road test that exercises the antilock system. Detailed removal, installation, and air-gap steps are provided in [Summit Abs Service Manual :: Wheel Speed Sensor Replacement Procedure](doc://man-summit-abs#wheel-speed-sensor-sub-system-service/wheel-speed-sensor-replacement-procedure) and [Summit Abs Service Manual :: Wheel Speed Sensor Sub-System Service](doc://man-summit-abs#wheel-speed-sensor-sub-system-service). For a comparable diagnostic reference, see [Falcon Abs Service Manual :: Wheel Speed Sensor Codes](doc://man-falcon-abs#diagnostic-trouble-codes/wheel-speed-sensor-codes).
