id: tsb-27-106
title: "Summit 5.0 V8: Left front wheel speed sensor circuit fault disables ABS"
area: bulletins
---

# Condition

The driver may report an illuminated ABS warning lamp on the Summit 5.0 V8 with DTC C0035 (Left Front Wheel Speed Sensor Circuit) stored in the EBCM. A companion code U0121 (Lost Communication With ABS Control Module) may also set on other modules when the EBCM enters a fault state and stops broadcasting wheel speed data. Antilock braking, traction control, and stability functions are disabled while these codes are active, though base braking remains available. Confirm the active codes against [Summit Abs Service Manual :: Wheel Speed Sensor Circuit Codes](doc://man-summit-abs#diagnostic-trouble-codes/wheel-speed-sensor-circuit-codes).

# Cause

C0035 on this platform is most commonly caused by an open or high-resistance circuit at the left front wheel speed sensor WSS-3300, often from connector corrosion or a chafed harness near the strut. Once the EBCM detects the invalid signal it disables the ABS hydraulic functions, which can in turn trigger downstream U0121 communication faults on the bus. Inspect sensor WSS-3300 and its harness per [Summit Abs Service Manual :: Wheel Speed Sensors](doc://man-summit-abs#abs-system-overview/wheel-speed-sensors) and review the network fault criteria in [Summit Abs Service Manual :: System Configuration and Communication Codes](doc://man-summit-abs#diagnostic-trouble-codes/system-configuration-and-communication-codes).

# Repair Procedure

Replace left front wheel speed sensor WSS-3300 following PROC-WSS as described in the [Summit Abs Service Manual :: Wheel Speed Sensor Replacement Procedure](doc://man-summit-abs#wheel-speed-sensor-sub-system-service/wheel-speed-sensor-replacement-procedure), ensuring the air gap and tone ring are clean and undamaged. After installation, clear C0035 and U0121, then verify the EBCM resumes normal communication using the [Summit Abs Service Manual :: ABS Module Diagnosis Procedure](doc://man-summit-abs#ebcm-and-hydraulic-sub-system-service/abs-module-diagnosis-procedure). Perform a road test above 25 mph to confirm all wheel speed signals track together, referencing the [Summit Abs Service Manual :: Wheel Speed Sensor Sub-System Service](doc://man-summit-abs#wheel-speed-sensor-sub-system-service) as needed. General system context is available in the [Summit Abs Service Manual :: ABS System Overview](doc://man-summit-abs#abs-system-overview).
