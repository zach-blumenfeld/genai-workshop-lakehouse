title: "Lumen 2.0T: EPS torque sensor fault causing intermittent steering assist loss"
area: bulletins
---

# Condition

Drivers of the Lumen 2.0T may report a sudden, intermittent loss of power steering assist, with the steering wheel becoming heavy for a few seconds before assist returns. The event is usually logged as DTC C1556 in the EPS controller and may be accompanied by the EPS warning lamp. Capture freeze-frame data at the time of the fault to aid diagnosis. See the [Lumen Steering Service Manual :: Steering System Overview](doc://man-lumen-steering#steering-system-overview) for normal assist behavior.

# Cause

DTC C1556 is set when the EPS controller detects an implausible or out-of-range signal from the torque sensor integrated into the electric power steering motor (EPS-8740). Internal contamination or a degraded sensor winding within the EPS-8740 assembly produces the intermittent dropout that triggers the temporary loss of assist. Reference the [Lumen Steering Service Manual :: EPS Torque Sensor Fault](doc://man-lumen-steering#diagnostic-trouble-codes/eps-torque-sensor-fault) for the code-setting criteria and the [Lumen Steering Service Manual :: Electric Power Steering Motor](doc://man-lumen-steering#steering-system-overview/electric-power-steering-motor) for component detail.

# Repair Procedure

Diagnose C1556 by verifying battery voltage, EPS connector integrity, and torque sensor signal per PROC-EPS-DIAG before replacing the electric power steering motor (EPS-8740) assembly. The torque sensor is not serviced separately, so a confirmed sensor fault requires replacing the complete EPS-8740 unit and performing the steering angle relearn afterward. Follow the full diagnostic flow in the [Lumen Steering Service Manual :: Electric Power Steering Diagnosis Procedure](doc://man-lumen-steering#electric-power-steering-sub-system-service/electric-power-steering-diagnosis-procedure), with the sub-system service context in the [Lumen Steering Service Manual :: Electric Power Steering Sub-System Service](doc://man-lumen-steering#electric-power-steering-sub-system-service). The complete code listing is in the [Lumen Steering Service Manual :: Diagnostic Trouble Codes](doc://man-lumen-steering#diagnostic-trouble-codes).
