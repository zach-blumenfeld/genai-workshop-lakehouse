# Condition

Affected Ridgeline HD vehicles equipped with the 6.7 TD diesel engine may illuminate the malfunction indicator lamp with DTC P2002 (Diesel Particulate Filter Efficiency Below Threshold Bank 1) stored. The condition typically presents after extended low-speed or short-trip operation that prevents the filter from reaching passive regeneration temperature. Customers may report reduced power, frequent regeneration cycles, or an elevated exhaust backpressure warning. For the full code definition and diagnostic entry point, refer to the [Ridgeline HD Emissions Service Manual :: Emissions Diagnostic Trouble Code Reference](doc://man-ridgeline-hd-emissions#diagnosis/emissions-diagnostic-trouble-code-reference).

# Cause

Investigation has determined that DTC P2002 on these vehicles is caused by progressive ash loading in diesel particulate filter assembly DPF-5180, which reduces measured filtration efficiency below the calibrated threshold. The condition is accelerated when passive regeneration is repeatedly interrupted, allowing soot and ash to accumulate beyond the self-cleaning capacity of DPF-5180. Begin diagnosis using [Ridgeline HD Emissions Service Manual :: Diagnosis](doc://man-ridgeline-hd-emissions#diagnosis) and cross-check filter behavior against [Kestrel Emissions Service Manual :: DPF Efficiency Faults](doc://man-kestrel-emissions#emissions-system-diagnosis/dpf-efficiency-faults). System background is described in [Ridgeline HD Emissions Service Manual :: Emissions System Overview](doc://man-ridgeline-hd-emissions#emissions-system-overview).

# Repair Procedure

Perform a forced regeneration on diesel particulate filter DPF-5180 following procedure PROC-DPF-REGEN before considering replacement, as many units recover acceptable efficiency once ash is converted and expelled. If P2002 returns after PROC-DPF-REGEN and ash load remains above limit, replace DPF-5180 per the service steps in [Ridgeline HD Emissions Service Manual :: Diesel Particulate Filter Service and Forced Regeneration](doc://man-ridgeline-hd-emissions#diesel-particulate-filter-sub-system/diesel-particulate-filter-service-and-forced-regeneration). General regeneration theory and tooling are documented in [Kestrel Emissions Service Manual :: DPF Service and Regeneration](doc://man-kestrel-emissions#diesel-particulate-filter/dpf-service-and-regeneration). Clear all codes and verify a completed regeneration cycle before returning the vehicle to the customer.
