# Condition

Customers may report intermittent loss of power steering assist accompanied by an illuminated EPS warning lamp on affected Lumen vehicles. When this condition is present, the steering control module sets diagnostic trouble code C1556, indicating a torque sensor fault. Assist may be reduced or absent until the ignition is cycled, and the code will store in current or history memory. For a description of the affected circuit, see the [Lumen Steering Service Manual :: Electric Power Steering Motor](doc://man-lumen-steering#steering-system-overview/electric-power-steering-motor).

# Cause

The C1556 fault is caused by an intermittent open in the torque sensor signal circuit internal to the steering column assembly, part number EPS-8740. Moisture intrusion at the sensor connector degrades the signal reference, causing the module to detect an implausible torque value and disable assist. The detailed code definition is documented in the [Lumen Steering Service Manual :: EPS Torque Sensor Fault](doc://man-lumen-steering#diagnostic-trouble-codes/eps-torque-sensor-fault). The equivalent condition on the related platform is covered in the [Osprey Steering Service Manual :: C1556 EPS Torque Sensor Fault](doc://man-osprey-steering#diagnostic-trouble-codes/c1556-eps-torque-sensor-fault).

# Repair Procedure

Replace the steering column torque sensor assembly EPS-8740, then clear C1556 and verify assist is restored through the full lock-to-lock range. Before replacement, confirm the fault is internal by following PROC-EPS-DIAG as described in the [Lumen Steering Service Manual :: Electric Power Steering Diagnosis Procedure](doc://man-lumen-steering#electric-power-steering-sub-system-service/electric-power-steering-diagnosis-procedure). After installation, recalibrate the sensor zero point per the [Lumen Steering Service Manual :: Electric Power Steering Sub-System Service](doc://man-lumen-steering#electric-power-steering-sub-system-service). General torque specifications and component locations are available in the [Lumen Steering Service Manual](doc://man-lumen-steering).
