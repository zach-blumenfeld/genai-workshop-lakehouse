# Condition

The Lumen 2.0T may exhibit loss of power steering assist accompanied by an illuminated EPS warning lamp and diagnostic trouble code C1556 stored in the steering control module. Drivers may report heavy steering effort, an off-center assist feel, or intermittent assist dropout during cold starts. Code C1556 identifies a fault in the electric power steering torque sensor signal. Review the [Lumen Steering Service Manual :: Steering System Overview](doc://man-lumen-steering#steering-system-overview) and the [Lumen Steering Service Manual :: Electric Power Steering Motor](doc://man-lumen-steering#steering-system-overview/electric-power-steering-motor) section to understand assist behavior under fault conditions.

# Cause

Code C1556 sets when the steering control module detects an implausible or out-of-range signal from the torque sensor integrated in the EPS assembly EPS-8740. Common contributors include connector corrosion, internal sensor degradation, and supply-voltage instability during high electrical load. A failed torque sensor within EPS-8740 cannot be serviced separately and requires assembly replacement. The fault isolation steps are detailed in the [Lumen Steering Service Manual :: EPS Torque Sensor Fault](doc://man-lumen-steering#diagnostic-trouble-codes/eps-torque-sensor-fault) and within the broader [Lumen Steering Service Manual :: Diagnostic Trouble Codes](doc://man-lumen-steering#diagnostic-trouble-codes) chapter.

# Repair Procedure

Perform EPS diagnosis per [PROC-EPS-DIAG] as documented in the [Lumen Steering Service Manual :: Electric Power Steering Diagnosis Procedure](doc://man-lumen-steering#electric-power-steering-sub-system-service/electric-power-steering-diagnosis-procedure). Confirm power and ground integrity before replacing the EPS assembly EPS-8740, and perform the required torque sensor calibration after installation. Reference the [Lumen Steering Service Manual :: Electric Power Steering Sub-System Service](doc://man-lumen-steering#electric-power-steering-sub-system-service) chapter for mounting torque and calibration steps.

# Verification

Clear C1556 with a scan tool and verify the code does not reset after a key cycle and a brief road test. Confirm assist is centered and smooth in both directions and that no related steering codes remain pending. Use the diagnostic confirmation steps in the [Lumen Steering Service Manual :: EPS Torque Sensor Fault](doc://man-lumen-steering#diagnostic-trouble-codes/eps-torque-sensor-fault) section to validate the repair before releasing the vehicle.
