# Condition

Some Harrier 3.5 V6 vehicles may set diagnostic trouble code P0011, indicating camshaft position timing is over-advanced on bank 1. Drivers may report rough idle, hesitation on light acceleration, and a momentary reduction in power with the MIL illuminated. The condition often appears after extended intervals on degraded engine oil. See the [Harrier Engine Service Manual :: Engine System Overview](doc://man-harrier-engine#engine-system-overview) for a description of the bank 1 camshaft phaser circuit.

# Cause

The cause is a stuck-open variable valve timing control solenoid, part number VVT-3360, which allows excessive oil flow to the bank 1 phaser and over-advances cam timing, setting P0011. Internal solenoid spool sticking is commonly the result of sludge or oil contamination restricting free travel. The [Harrier Engine Service Manual :: Timing Sub-System](doc://man-harrier-engine#timing-sub-system) describes the phaser oil control logic, and the closely related [Falcon Engine Service Manual :: Variable Valve Timing System](doc://man-falcon-engine#variable-valve-timing-system) and [Lumen Engine Service Manual :: Valve Timing Sub-System](doc://man-lumen-engine#valve-timing-sub-system) cover equivalent solenoid behavior on sister platforms.

# Repair Procedure

Replace the VVT control solenoid, part number VVT-3360, following the [Harrier Engine Service Manual :: Variable Valve Timing Solenoid Replacement](doc://man-harrier-engine#timing-sub-system/variable-valve-timing-solenoid-replacement) (PROC-VVT-SOL). Perform an engine oil and filter change (PROC-OIL) at the same time, since contaminated oil is the most frequent contributor to solenoid sticking. After installation, clear codes and verify P0011 does not reset under load during a road test. Equivalent solenoid service steps are documented in the [Falcon Engine Service Manual :: VVT Solenoid Replacement](doc://man-falcon-engine#variable-valve-timing-system/vvt-solenoid-replacement) and the [Lumen Engine Service Manual :: Variable Valve Timing Solenoid Service](doc://man-lumen-engine#valve-timing-sub-system/variable-valve-timing-solenoid-service) for technicians cross-trained on those platforms.
