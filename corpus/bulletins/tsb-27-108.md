# Condition

Affected Vega EV-180 vehicles may exhibit intermittent or total loss of communication with the body control module, setting DTC U0140 (Lost Communication with Body Control Module). Symptoms include inoperative interior lighting, non-responsive lock and unlock functions, and an instrument cluster that fails to display body status messages. Refer to [Network Communication Diagnostic Codes](doc://man-vega-electrical#network-communication-diagnostic-codes) for the full code description and the [Lost Communication with BCM and Cluster](doc://man-vega-electrical#network-communication-diagnostic-codes/lost-communication-with-bcm-and-cluster) subsection for symptom correlation before proceeding.

# Cause

This condition is caused by an internal microcontroller fault in body control module BCM-9920 that intermittently drops the module off the high-speed CAN bus, triggering U0140 at the gateway and dependent nodes. The fault is heat-sensitive and may not reproduce until the module reaches operating temperature. Background on the affected component and its bus role is provided in the [Body Control Module](doc://man-vega-electrical#electrical-system-overview/body-control-module) topic within the [Electrical System Overview](doc://man-vega-electrical#electrical-system-overview).

# Repair Procedure

First confirm bus integrity by running PROC-COMMS-DIAG and the [CAN Bus Communication Diagnosis Procedure](doc://man-vega-electrical#network-and-harness-sub-system-service/can-bus-communication-diagnosis-procedure) to rule out wiring and termination faults before condemning the module. If the bus is healthy, replace body control module BCM-9920 and complete configuration per PROC-BCM-CONFIG using the [Body Control Module Configuration Procedure](doc://man-vega-electrical#bcm-sub-system-service/body-control-module-configuration-procedure). Additional service context for the module is documented under [BCM Sub-System Service](doc://man-vega-electrical#bcm-sub-system-service). After repair, clear U0140 and verify communication is restored across all body functions.
