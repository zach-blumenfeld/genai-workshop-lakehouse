id: tsb-21-114
title: Revised Ignition Coil for Repeated Misfire
area: bulletins
---

# Condition

Some 2019-2021 Falcon vehicles with the 2.0T engine exhibit repeat misfire
codes P0301 or P0300 after a coil has already been replaced. For the baseline
diagnostic routine, see [Misfire Diagnosis](doc://man-fal-3#engine/misfire-diagnosis).

# Cause

The original ignition coil IC-2042-A insulation can degrade under repeated heat
cycles, causing intermittent misfire that returns after replacement with the
same part.

# Repair Procedure

Replace all coils with the revised ignition coil IC-2042-B, which supersedes
IC-2042-A, following
[Ignition Coil Replacement](doc://man-fal-3#engine/ignition-system/ignition-coil-replacement).
Replace spark plug set SP-1108 if beyond half service life. The connector
failure mode is also addressed by
[safety recall RC-2021-04](doc://rc-2021-04).
