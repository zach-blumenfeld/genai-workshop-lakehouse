title: Summit left front wheel speed sensor circuit fault triggering ABS warning
area: bulletins
---

# Condition

Affected Summit vehicles may set DTC C0035 (Left Front Wheel Speed Sensor Circuit) with the ABS warning lamp illuminated and the traction control disabled. Drivers typically report the warning appears intermittently, often after driving over rough roads or in wet conditions. The wheel speed sensor at the affected corner, part number WSS-3300, reports an invalid or dropped signal that triggers C0035. Sensor operation and circuit layout are described in the [Summit Abs Service Manual :: Wheel Speed Sensors](doc://man-summit-abs#abs-system-overview/wheel-speed-sensors).

# Cause

The fault is caused by a chafed or corroded connector at the left front WSS-3300, which interrupts the sensor signal and sets C0035. Connector pin corrosion and harness routing near the strut are the most common contributors. A failed sensor element itself is less common but should be confirmed during diagnosis. Circuit code definitions are listed in the [Summit Abs Service Manual :: Wheel Speed Sensor Circuit Codes](doc://man-summit-abs#diagnostic-trouble-codes/wheel-speed-sensor-circuit-codes).

# Diagnosis

Follow procedure PROC-ABS-DIAG to isolate whether C0035 originates from the WSS-3300 sensor, its connector, or the harness before replacing parts. Inspect the connector for corrosion and back-probe the circuit while wiggling the harness to reproduce the intermittent signal loss. If circuit integrity is good, evaluate the module side using the [Summit Abs Service Manual :: ABS Module Diagnosis Procedure](doc://man-summit-abs#ebcm-and-hydraulic-sub-system-service/abs-module-diagnosis-procedure). General code reference is in the [Summit Abs Service Manual :: Diagnostic Trouble Codes](doc://man-summit-abs#diagnostic-trouble-codes).

# Repair Procedure

When the sensor or its connector is confirmed faulty, replace the WSS-3300 per procedure PROC-WSS, ensuring the mounting surface is clean and the air gap is correct. Route the new sensor harness exactly as original and secure all retainers to prevent recurrence of C0035. Clear codes and confirm all four wheel speed signals track together during a road test. Detailed installation steps are in the [Summit Abs Service Manual :: Wheel Speed Sensor Replacement Procedure](doc://man-summit-abs#wheel-speed-sensor-sub-system-service/wheel-speed-sensor-replacement-procedure).
