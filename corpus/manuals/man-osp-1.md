# Engine

Service coverage for the Cascadia Osprey 3.5 V6.

## Misfire Diagnosis

The 3.5 V6 sets P0300 for random misfire across banks. Coil-on-plug ignition is
shared in concept with the Falcon; the diagnostic flow in the
[Falcon Misfire Diagnosis](doc://man-fal-3#engine/misfire-diagnosis) applies,
but the Osprey uses a different coil and the SP-1108 spark plug set is not
compatible. Confirm both banks before condemning components.

## Catalyst Efficiency

P0420 on the V6 is bank-specific. As with the Falcon, rule out misfire before
replacing CAT-5500; an ECM-9901 update may apply
(see [Updated Catalyst Monitor Calibration](doc://tsb-20-087)).

# Brakes

## Front Brake Service

The Osprey shares the BP-7720 front pad kit and BR-7731 rotor pair with the
Falcon. The judder bulletin
[Brake Judder After Front Pad Replacement](doc://tsb-22-031) applies to both
platforms.

# ABS

## Wheel Speed Sensors

The Osprey uses the same WSS-3300 wheel speed sensor. Code C0035 and the harness
routing concern in [Wheel Speed Sensor Harness Chafing](doc://rc-2022-09) apply.
