# Defect Description

Affected vehicles were built with EVAP canister purge valve EVP-5160, in which a defective internal solenoid seat may cause the valve to stick open. A stuck-open purge valve allows fuel vapor to escape the sealed evaporative system, setting the gross leak code P0455 and, on smaller intrusions, the small leak code P0442. The uncontrolled release of fuel vapor raises emissions and, in confined or poorly ventilated conditions, presents a vapor exposure concern. Diagnosis of the leak should follow [Kestrel Emissions Service Manual :: EVAP Leak Faults](doc://man-kestrel-emissions#emissions-system-diagnosis/evap-leak-faults) and the broader procedure in [Kestrel Emissions Service Manual :: Emissions System Diagnosis](doc://man-kestrel-emissions#emissions-system-diagnosis), with code definitions confirmed against [Ridgeline HD Emissions Service Manual :: Emissions Diagnostic Trouble Code Reference](doc://man-ridgeline-hd-emissions#diagnosis/emissions-diagnostic-trouble-code-reference).

# Remedy

Dealers will replace the affected EVP-5160 purge valve with a redesigned valve having an improved solenoid seat, free of charge. The replacement must follow the service steps in [Kestrel Emissions Service Manual :: EVAP Purge Valve Service](doc://man-kestrel-emissions#evaporative-emission-control/evap-purge-valve-service) within the [Kestrel Emissions Service Manual :: Evaporative Emission Control](doc://man-kestrel-emissions#evaporative-emission-control) system. After installation, perform an evaporative system leak test per [Ridgeline HD Emissions Service Manual :: EVAP Leak Diagnosis](doc://man-ridgeline-hd-emissions#diagnosis/evap-leak-diagnosis) and confirm that P0455 and P0442 do not reset. Owners can verify recall status at https://www.nhtsa.gov/recalls.
