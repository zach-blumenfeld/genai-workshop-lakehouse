title: Heron 2.5 ignition coil internal short can cause misfire and stall

# Defect Description

On affected Heron 2.5 vehicles, the application-specific ignition coil IC-2044-A may develop an internal short between the primary and secondary windings after prolonged thermal cycling. As the insulation degrades, the coil delivers an intermittent or absent spark to the affected cylinder, setting cylinder-specific misfire code P0351 and, in severe cases, the random/multiple misfire code P0300. The resulting misfire can cause rough running, illumination of the malfunction indicator lamp, and an unexpected engine stall while driving. Diagnostic confirmation should follow the procedures in the [Heron Ignition Service Manual :: Coil Primary and Secondary Circuit Faults](doc://man-heron-ignition#ignition-circuit-diagnosis/coil-primary-and-secondary-circuit-faults), with supporting background in [Heron Ignition Service Manual :: Application-Specific Coils](doc://man-heron-ignition#ignition-coils/application-specific-coils). Technicians may also reference the [Falcon Engine Service Manual :: Misfire Diagnosis](doc://man-falcon-engine#misfire-diagnosis) for comparable P0300-family symptom patterns.

# Remedy

Authorized dealers will replace the affected IC-2044-A ignition coil with a revised coil using upgraded winding insulation, free of charge, following procedure PROC-IGN-COIL. Perform the replacement per the steps in the [Heron Ignition Service Manual :: Ignition Coil Replacement Procedure](doc://man-heron-ignition#ignition-coils/ignition-coil-replacement-procedure), then verify spark integrity using [Heron Ignition Service Manual :: Ignition Circuit Diagnosis](doc://man-heron-ignition#ignition-circuit-diagnosis). After installation, clear stored P0351 and P0300 codes and confirm no misfire recurs during a road test; consult the [Harrier Engine Service Manual :: Misfire Diagnostic Trouble Code Reference](doc://man-harrier-engine#diagnosis/misfire-diagnostic-trouble-code-reference) if related codes remain active. Owners should contact their dealer to schedule this no-charge repair.
