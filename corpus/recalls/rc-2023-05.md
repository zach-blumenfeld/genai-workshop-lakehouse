# Defect Description

On affected Summit vehicles, the electronic brake control module EBCM-8100 may contain a pump relay with a marginal solder joint that opens under vibration and thermal stress. When the relay circuit fails, the module sets pump motor circuit code C0265 and may also log control module communication code C0561 as the EBCM drops off the network. The anti-lock braking and stability control functions are then disabled, although base hydraulic braking is retained. Diagnostic confirmation should follow [Summit Abs Service Manual :: Pump and Relay Circuit Codes](doc://man-summit-abs#diagnostic-trouble-codes/pump-and-relay-circuit-codes), with module background in [Summit Abs Service Manual :: Electronic Brake Control Module (EBCM)](doc://man-summit-abs#abs-system-overview/electronic-brake-control-module-ebcm). Technicians may cross-reference the [Falcon Abs Service Manual :: Pump and Relay Codes](doc://man-falcon-abs#diagnostic-trouble-codes/pump-and-relay-codes) for comparable C0265 patterns.

# Safety Risk and Affected Vehicles

When code C0561 is set and the EBCM-8100 drops offline, the driver loses ABS and stability control without warning, increasing the risk of wheel lockup and loss of directional control during hard or low-traction braking. Affected vehicles are Summit units built within the identified production window using modules from the suspect assembly lot. Drivers may notice ABS and traction warning lamps illuminated together. Communication-related faults should be evaluated per [Summit Abs Service Manual :: System Configuration and Communication Codes](doc://man-summit-abs#diagnostic-trouble-codes/system-configuration-and-communication-codes).

# Remedy

Authorized dealers will replace the affected EBCM-8100 module with a revised unit using an upgraded pump relay, free of charge, following procedure PROC-ABS-DIAG. Perform module diagnosis and replacement per [Summit Abs Service Manual :: ABS Module Diagnosis Procedure](doc://man-summit-abs#ebcm-and-hydraulic-sub-system-service/abs-module-diagnosis-procedure), then verify relay operation and clear any stored C0265 codes. Confirm ABS and stability control restore full function during a road test before returning the vehicle; consult [Summit Abs Service Manual :: EBCM and Hydraulic Sub-System Service](doc://man-summit-abs#ebcm-and-hydraulic-sub-system-service) if related faults persist. Owners should contact their dealer to schedule this no-charge repair.
