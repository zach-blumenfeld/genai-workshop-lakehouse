title: Safety Recall - Alternator Output Failure Stall While Driving
area: recalls
---

# Defect Description

Affected vehicles were built with alternator assembly ALT-8810 units containing a defective voltage regulator that can intermittently stop charging output. When output collapses, the powertrain control module detects undervoltage and sets diagnostic trouble code P0562, which may be accompanied by charge warning lamps. Technicians should confirm the condition using the [Falcon Charging Service Manual :: Charging System Diagnosis Procedure](doc://man-falcon-charging#charging-system-diagnosis-sub-system/charging-system-diagnosis-procedure) and cross-check the code against the [Falcon Charging Service Manual :: System Voltage Codes](doc://man-falcon-charging#diagnostic-trouble-codes/system-voltage-codes) reference. Detail on the failing component is found in the [Falcon Charging Service Manual :: Alternator](doc://man-falcon-charging#charging-system-overview/alternator) section.

# Safety Risk

With ALT-8810 not charging, the vehicle runs solely on stored energy in the battery assembly BAT-1200, which depletes within minutes and leads to an engine stall while driving. A stall at speed reduces power steering and braking assist, increasing crash risk. The [Falcon Charging Service Manual :: Battery](doc://man-falcon-charging#charging-system-overview/battery) section describes how BAT-1200 sustains loads during a charging fault. A weak or aged battery shortens the available run time before the stall occurs.

# Remedy

Dealers will replace the alternator assembly ALT-8810 with a revised unit using an updated voltage regulator, following the diagnosis flow in the [Falcon Charging Service Manual :: Charging System Overview](doc://man-falcon-charging#charging-system-overview). Where the battery assembly BAT-1200 has been deeply discharged by repeated stalls, it must be tested and, if it fails, replaced per the [Falcon Charging Service Manual :: Battery Replacement Procedure](doc://man-falcon-charging#battery-sub-system-service/battery-replacement-procedure). After repair, verify P0562 does not reset and review the [Falcon Charging Service Manual :: Generator Control Circuit Code](doc://man-falcon-charging#diagnostic-trouble-codes/generator-control-circuit-code) for related fault confirmation. This remedy is performed at no cost to owners; see https://www.nhtsa.gov/recalls for notification timing.
