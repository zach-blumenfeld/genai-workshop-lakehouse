# Defect Description

On certain Falcon and Osprey vehicles, the front wheel speed sensor WSS-3300
harness can chafe against the brake hose BH-7745 routing, setting code C0035 and,
if it shorts, ABS communication loss U0121.

# Remedy

Dealers will inspect and re-route the WSS-3300 harness and replace it if chafed,
at no charge. Diagnose any stored faults per
[ABS Module Diagnosis](doc://man-fal-3#abs/abs-module-diagnosis) and
[Wheel Speed Sensors](doc://man-fal-3#abs/wheel-speed-sensors). Owners can check
recall status at [NHTSA](https://www.nhtsa.gov/recalls).
