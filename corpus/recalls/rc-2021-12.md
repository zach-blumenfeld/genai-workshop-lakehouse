title: Safety Recall - Shift Solenoid Pack Failure Unintended Gear Engagement
area: recalls

# Defect Description

Affected transmissions were assembled with shift solenoid pack TSOL-9520 units that can stick due to a manufacturing burr in the valve bore. When a solenoid hangs, the transmission control module sets diagnostic trouble code P0700 along with the specific shift fault code P0741 indicating a torque converter clutch circuit performance issue. Technicians should confirm these codes using the [Osprey Transmission Service Manual :: Transmission Fault Diagnosis Procedure](doc://man-osprey-transmission#diagnosis/transmission-fault-diagnosis-procedure) and the [Osprey Transmission Service Manual :: Diagnostic Trouble Code Reference](doc://man-osprey-transmission#diagnosis/diagnostic-trouble-code-reference). A related pressure symptom can be isolated with the [Osprey Transmission Service Manual :: Fluid Pressure Diagnosis](doc://man-osprey-transmission#diagnosis/fluid-pressure-diagnosis) procedure.

# Safety Risk

A stuck solenoid can command an unintended gear engagement, producing an abrupt change in driveline behavior that the driver does not expect. This can cause unexpected vehicle movement or loss of acceleration control, increasing crash risk. The torque converter clutch behavior associated with this fault is described in the [Osprey Transmission Service Manual :: Torque Converter Clutch Diagnosis](doc://man-osprey-transmission#shift-control-sub-system/torque-converter-clutch-diagnosis) section. Drivers should not continue operating the vehicle if harsh or unexpected shifts are observed.

# Remedy

Dealers will replace the shift solenoid pack TSOL-9520 with a revised assembly machined to corrected tolerances, following the [Osprey Transmission Service Manual :: Shift Solenoid Pack Replacement Procedure](doc://man-osprey-transmission#shift-control-sub-system/shift-solenoid-pack-replacement-procedure). After replacement, the technician should clear codes and verify normal operation against the baseline in the [Osprey Transmission Service Manual :: Transmission System Overview](doc://man-osprey-transmission#transmission-system-overview). This remedy is provided free of charge; owner notification details are available at https://www.nhtsa.gov/recalls.
