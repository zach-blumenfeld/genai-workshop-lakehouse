title: Safety Recall - Ignition Coil Internal Arcing and Stall
area: recalls
---

# Defect Description

Affected vehicles were built with ignition coil assembly IC-2044-A, in which a manufacturing defect in the secondary winding insulation allows internal arcing under high-load conditions. The arcing produces an intermittent loss of spark on the affected cylinder, setting cylinder-specific misfire code P0351 and, as the condition spreads, the random/multiple misfire code P0300. A severe misfire event can cause a rough idle, hesitation, or an unexpected engine stall while driving, which increases the risk of a crash. Diagnosis of the underlying spark loss should follow the procedures in [Falcon Engine Service Manual :: Misfire Diagnosis](doc://man-falcon-engine#misfire-diagnosis) and the coil-level checks in [Heron Ignition Service Manual :: Coil Primary and Secondary Circuit Faults](doc://man-heron-ignition#ignition-circuit-diagnosis/coil-primary-and-secondary-circuit-faults). Technicians should confirm the freeze-frame data against the [Harrier Engine Service Manual :: Misfire Diagnostic Trouble Code Reference](doc://man-harrier-engine#diagnosis/misfire-diagnostic-trouble-code-reference) before condemning the coil.

# Remedy

Dealers will replace all affected IC-2044-A ignition coils with a redesigned coil featuring improved secondary winding insulation, free of charge. The replacement must follow the removal and torque sequence described in [Heron Ignition Service Manual :: Application-Specific Coils](doc://man-heron-ignition#ignition-coils/application-specific-coils), with verification of correct connector seating per [Heron Ignition Service Manual :: Ignition Coils](doc://man-heron-ignition#ignition-coils). After installation, clear stored codes and road-test to confirm no recurrence of P0351 or P0300, referencing [Harrier Engine Service Manual :: Misfire Diagnosis Procedure](doc://man-harrier-engine#diagnosis/misfire-diagnosis-procedure). Owners can confirm recall applicability and remedy status at https://www.nhtsa.gov/recalls.
