# Defect Description

Certain Falcon vehicles were built with an alternator, part number ALT-8810, whose generator control circuit can experience an internal regulator fault. When the fault occurs the PCM sets diagnostic trouble code P0620 for the generator control circuit, and the resulting voltage instability can also set system voltage code P0562. The alternator is described in the [Falcon Charging Service Manual :: Alternator](doc://man-falcon-charging#charging-system-overview/alternator), with the relevant codes detailed under [Falcon Charging Service Manual :: Generator Control Circuit Code](doc://man-falcon-charging#diagnostic-trouble-codes/generator-control-circuit-code) and [Falcon Charging Service Manual :: System Voltage Codes](doc://man-falcon-charging#diagnostic-trouble-codes/system-voltage-codes).

# Safety Risk and Affected Vehicles

A control fault in ALT-8810 can cause the charging system to stop supplying current, leading to a sudden engine stall and loss of power steering and braking assist while driving, which increases crash risk. Affected vehicles are Falcon units within the identified build window; confirm the charging configuration using the [Falcon Charging Service Manual](doc://man-falcon-charging) and the [Falcon Charging Service Manual :: Charging System Overview](doc://man-falcon-charging#charging-system-overview) before service.

# Remedy

Dealers will replace the alternator ALT-8810 with a revised unit free of charge following procedure PROC-ALT, then clear P0620 and P0562 and verify output. Perform the diagnosis using the [Falcon Charging Service Manual :: Charging System Diagnosis Procedure](doc://man-falcon-charging#charging-system-diagnosis-sub-system/charging-system-diagnosis-procedure) and complete the install per the [Falcon Charging Service Manual :: Alternator Replacement Procedure](doc://man-falcon-charging#alternator-sub-system-service/alternator-replacement-procedure). Owners may verify recall status and schedule the remedy at https://www.nhtsa.gov/recalls.
